# Missing Landing Page

Missing — одностраничный лендинг сервиса релокации, который объясняет процесс переезда: этапы, направления и тарифы.
Проект сделан с упором на аккуратный UI, темизацию и понятную архитектуру фронтенд-приложения.

Лендинг ориентирован на пользователя, который впервые сталкивается с релокацией и хочет быстро понять:

- как устроен процесс,
- какие шаги его ждут,
- какие есть варианты и стоимость.

---

## О проекте

Цель проекта - показать:

- умение работать с компонентной архитектурой React,
- осознанное использование CSS variables и темизации,
- аккуратную адаптивную вёрстку без UI-библиотек,
- чистую и объяснимую структуру проекта.

Проект не перегружен бизнес-логикой и абстракциями — решения подобраны под формат лендинга.

---

## Технологии

- React 18 + TypeScript
- Storybook (для UI-компонентов)
- Vite
- CSS Modules

---

## Ключевые решения

- Темизация реализована через CSS variables и state в App без Context API,
  так как проект является лендингом и не требует сложного глобального состояния
- Использована компонентная архитектура с разделением UI и фич
- CSS Modules для изоляции стилей 
- Адаптивная верстка с использованием CSS Grid и Flexbox с фокусом на удобство использования на мобильных устройствах.

---


## Функционал

- Header с навигацией и переключением темы

- Hero, Steps и Pricing секции

- Светлая и тёмная тема

- Hover / active состояния для интерактивных элементов

- Адаптивная вёрстка под мобильные устройства

- Переиспользуемые UI-компоненты

## Структура проекта

pages/ — страницы приложения
Отвечают за композицию виджетов и фич.

widgets/ — крупные блоки интерфейса
(Header, секции лендинга), собирают внутри себя UI и фичи.

shared/ — переиспользуемые UI-компоненты, стили и утилиты
(Button, layout-элементы, базовые стили).

public/ — статические файлы

styles/ — глобальные стили и переменные

## Дополнительно

- Сохранение темы (light/dark) через localStorage
- Адаптивная вёрстка под мобильные устройства

## Установка и запуск

1. Клонировать репозиторий:

```bash
git clone https://github.com/petgear/Missing-Landing.git
```

2. Перейти в директорию проекта:
```bash
cd Missing-Landing
```

3. Установить зависимости:
```bash
npm install
```

4. Запустить локальный сервер:
```bash
npm run dev
```
